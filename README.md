# Bank Management System

Small CLI-based bank management prototype. Features:

- Customer and admin management
- Accounts: Savings, Current, Loan
- JSON-based persistence in `data/bank_data.json`

Run:

```bash
python main.py
```

Notes:
- This repo is a learning prototype. Do not use in production (PINs stored as plaintext).
